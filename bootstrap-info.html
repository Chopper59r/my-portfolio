Bootstrap Features Used

This page summarises the Bootstrap 5 components, layout features, and utilities used in my personal portfolio website.

All layout and styling are implemented by combining Bootstrap with my own custom CSS and JavaScript.

Bootstrap Components

The following Bootstrap components are used throughout the portfolio to structure content and create a clean, responsive design.

Navbar

Implemented using the .navbar, .navbar-expand-lg, .navbar-dark, .bg-dark, and .fixed-top classes to create a responsive, dark navigation bar at the top of the page.

Includes a collapsible menu using .navbar-toggler, .navbar-toggler-icon, and .collapse.navbar-collapse.

Cards

Used to display content such as education details, project summaries, work placement information, and hobby descriptions. Implemented with .card, .card-body, .card-title, and .card-text.

Some cards also use .shadow-sm and .h-100 to add subtle shadows and equal heights.

Tables

A table is used in the Education section to list modules and grades. This uses .table, .table-striped, and .table-responsive to keep it readable on all devices.

Table headings use .table-light for contrast.

Progress Bars

The Skills & Achievements section uses Bootstrap progress bars to visually represent proficiency levels in different technical skills.

Implemented using .progress and .progress-bar, with inline style="width: X%;" to indicate the skill level.

List Group

Soft skills and achievements are displayed using .list-group and .list-group-item to keep the information structured and easy to read.

Some items also include .d-flex and .justify-content-between for aligned labels and badges.

Badges

Badges are used in the Projects section to indicate the year of each project (e.g. 2nd Year, 3rd Year, 4th Year).

Implemented using .badge with contextual colour classes such as .bg-primary, .bg-success, and .bg-warning.

Forms

The Contact section includes a simple contact form. Form fields use .form-label and .form-control to apply consistent Bootstrap styling.

The form fields include text inputs and a textarea to collect a name, email, subject, and message.

Modal

A Bootstrap modal is used to display full contact information when the user clicks the "View Full Contact Info" button.

Implemented with .modal, .modal-dialog, .modal-content, .modal-header, .modal-body, and .modal-footer, along with data-bs-toggle="modal" and data-bs-target="#contactModal".

Buttons

Various Bootstrap buttons are used throughout the site, including primary, outline, and small buttons.

Examples include .btn, .btn-primary, .btn-outline-primary, .btn-secondary, and .btn-sm.

Layout & Utility Classes

The site makes extensive use of Bootstrap's layout system and utility classes to create a responsive design without writing large amounts of custom CSS.

Layout & Grid

  • .container to centre and constrain content width.
  • .row and various .col- classes (e.g. .col-md-6, .col-lg-4) to create responsive grids for sections like Projects, Skills, Experience, and Hobbies.
  • .g-4 to add consistent gutters (spacing) between columns.
  • .order-* classes (e.g. .order-1, .order-md-2) in the hero section to control the order of text and image on different screen sizes.

Utility Classes

  • Spacing: .py-5, .mb-3, .mb-4, .mt-4, .me-2, etc., to control margins and padding.
  • Typography: .display-4, .fw-bold, .lead, .h5, .h6, .text-muted, and .text-center.
  • Background & borders: .bg-light, .bg-white, .bg-dark, .border-start, .border-2, .border-primary.
  • Images & shapes: .img-fluid for responsive images and .rounded-circle for circular profile images.
  • Flexbox: .d-flex, .justify-content-between, .align-items-center, and .mt-auto to control alignment inside cards and list items.
  • Shadows: .shadow-sm to add subtle drop shadows to cards and images.
  • Visibility: .d-none is used dynamically by custom JavaScript to hide and show project cards during filtering.

JavaScript & Interactivity

The site uses both Bootstrap's JavaScript bundle and my own custom JavaScript file to provide interactive features.

Bootstrap JavaScript

  • The Bootstrap JavaScript bundle (including Popper) is loaded via CDN using bootstrap.bundle.min.js.
  • This is used for:
    • Navbar collapse / expansion on small screens.
    • Opening and closing the contact information modal.

Custom JavaScript (main.js)

  • A separate JavaScript file, assets/js/main.js, contains my own custom code.
  • Project filtering:
    • Clicking the filter buttons (All / 2nd Year / 3rd Year / 4th Year) shows or hides project cards by adding/removing the .d-none class based on a data-year attribute.
    • The active filter button is highlighted by toggling the .active class.
  • Contact form behaviour:
    • The contact form submission is intercepted with JavaScript to prevent a full page reload.
    • A friendly alert() message is displayed to the user, and the form fields are reset.
